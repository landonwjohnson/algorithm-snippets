# Relative Ranks

You are given an integer array `score` of size `n`, where `score[i]` is the score of the `i`th athlete in a competition. All scores are unique.

Return an array `answer` of size `n` where `answer[i]` is the rank string of the `i`th athlete:

- Rank `"Gold Medal"` for the 1st place score
- Rank `"Silver Medal"` for the 2nd place score
- Rank `"Bronze Medal"` for the 3rd place score
- For the 4th place and below, rank is the place number as a string (`"4"`, `"5"`, …)

The answer array must preserve the original order of athletes by index.
